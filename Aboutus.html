ABOUT US
Roblox is an online game platform and game creation system developed by Roblox Corporation that allows users to program games and play games created by other users. Created by David Baszucki and Erik Cassel in 2004 and released in 2006, the platform hosts user-created games of multiple genrecoded in the programming language Lua.
Roblox Studio is a development program developed by Roblox offering an assortment of coding and building tools for developing games for Roblox via the Lua programming language.
